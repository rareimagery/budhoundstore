## Useful Notes for Drupal Commerce Setup

When building out a Drupal Commerce store for any of these dispensaries, consider:

- **Store entity**: One store per dispensary location; use Commerce Multi-Store if managing multiple
- **Product catalog**: Use the product types defined in the Drupal Commerce Cannabis Products guide (flower, pre-rolls, edibles, vapes, concentrates, topicals, tinctures)
- **Age verification**: Implement an age gate using the `age_verification` module or a custom block — required for all dispensaries
- **Delivery zones**: Map each dispensary's delivery coverage using Commerce Shipping zones
- **Tax configuration**: California imposes a 15% cannabis excise tax + local sales tax; configure via Commerce Tax module. Note: Leaf Dispensary displays tax-inclusive prices — this requires a custom price display strategy
- **Loyalty programs**: Integrate with Commerce Points or a third-party loyalty platform (Alpine IQ, Springbig, etc.)
- **Compliance fields**: Ensure all product variations include THC%, CBD%, COA PDF upload, and METRC/BioTrack tag fields as outlined in the product types guide
